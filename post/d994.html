<3>下标

下标 方法适用于 字符串、列表、元组

所谓 “下标” ，就是编号，就好比超市中的存储柜的编号，通过这个编号就能找到相应的存储空间

<4>切片

切片 使用 索引值 来限定范围，从一个大的 字符串 中 切出 小的 字符串

切片的语法：[起始:结束:步长]

左闭右开区间一样: [ ) 大于等于 小于不等于

print(name[0:3]) 下标0~2的字符

print(name[1:-1])下标为一开始到最后第二

个之间的字符.包括最后第二个

<5>字符串常用方法

find

检测 str 是否包含在 mystr中，如果是返回开始的索引值，否则返回-1

index

跟find()方法一样，只不过如果str不在 mystr中会报一个异常.

count

返回 str在mystr里面出现的次数

replace

把 mystr 中的 str1 替换成 str2

split

以 - 为分隔符切片 mystr

join

a中每个元素后面插入-构造出一个新的字符串

capitalize

把字符串的第一个字符大写

title

把字符串的每个单词首字母大写 需要空格来识别

startswith

检查字符串是否是以 hello 开头, 是则返回 True，否则返回 False

endswith

检查字符串是否以obj结束，如果是返回True,否则返回 False.

lower

转换 mystr 中所有大写字符为小写

upper

转换 mystr 中的小写字母为大写

ljust

返回一个原字符串左对齐,并使用空格填充至长度 width 的新字符串

rjust

返回一个原字符串右对齐,并使用空格填充至长度 width 的新字符串

center

返回一个原字符串居中,并使用空格填充至长度 width 的新字符串

lstrip

删除 mystr 左边的空白字符

rstrip

删除 mystr 字符串末尾的空白字符

strip

删除mystr字符串两端的空白字符

rfind

类似于 find()函数，不过是从右边开始查找.

rindex

类似于 index()，不过是从右边开始.

isalpha

如果 mystr 所有字符都是字母 则返回 True,否则返回 False

isdigit
